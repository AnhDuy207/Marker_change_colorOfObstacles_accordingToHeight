```
https://github.com/AnhDuy207/Marker_change_colorOfObstacles_accordingToHeight.git
```

## Getting started
1. Run ROS Master
```
roscore
```
2. Run package  
```
source marker_ws/devel/setup.bash
rosrun using_markers basic_shapes 
```
3. Open rviz to check result
```
rviz
```
